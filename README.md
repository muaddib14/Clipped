# Clipped Landing Page

Landing page for Clipped — extract articles to markdown for Claude.

## Tech Stack

- Next.js 14
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Connected to Vercel. Push to main → auto deploy.

```bash
vercel
```

## Colors

- Primary: `#7c71e8` (purple)
- Cream: `#f5f0e8`
- Orange: `#ff9f43`
- Dark: `#0f0f0f`
